📖 How to Use — Step by Step

1

Download a Template or Sample File

Go to the Templates or Sample Files section below. Download the Excel file that best matches your bridge type. Templates have all required parameters pre-filled with standard values. Samples are real project inputs you can study and modify.

2

Edit the Excel File

Open the downloaded .xlsx file in Excel or LibreOffice Calc. The file has three columns: Value, Variable, Description. Edit only the Value column (column A). Do not rename or reorder the Variable column. Key parameters to set:

  • NSPAN — Number of spans (1, 2, 3…)
  • SPAN1 — Span length in metres
  • CCBR — Carriageway width in metres
  • DATUM — Datum level (reduced level of river bed)
  • TOPRL — Top of road level
  • SKEW — Skew angle in degrees (0 for straight)
3

Open the App and Upload

Click Open App ↗ to launch the Streamlit interface. Go to the Drawing tab (Tab 1). Click Browse files and upload your edited Excel file. The app will instantly show a parameter preview and bridge stats (spans, span length, width).

4

Run Quality Check (Optional but Recommended)

Switch to the Quality tab (Tab 4) after uploading. The IRC/IS compliance checker will score your design out of 100 and flag issues like insufficient vertical clearance (<5.5m per IRC 5:2015), slab thickness violations (L/20 rule), or narrow carriageway width.

5

Generate the Drawing

Back in Tab 1, select your AutoCAD version (R2010 recommended, R2006 for older software) and output format (DXF, PDF, PNG, SVG…). Click Generate Drawing. Generation takes 1–3 seconds. A download button appears immediately when ready.

6

Explore Other Tabs

Tab 2 — Bill: Generate statutory contractor bills with hierarchical items.
Tab 3 — Templates: Download any of 5 standard bridge templates as Excel.
Tab 5 — 3D: Interactive Plotly 3D model of your bridge.
Tab 7 — AI: Cost estimation and material quantity predictions.
Tab 8 — Export: Batch export to Excel, CSV, HTML, ZIP.

7

Batch Processing

In Tab 3, scroll to the Batch Processing section. Upload multiple Excel files at once. Click Generate All. All DXF files are bundled into a single ZIP for download.

8

API Usage (Developers)

POST your Excel file directly to the REST API and receive a DXF back. No browser needed — integrate into your own pipeline.

# Generate DXF via API
curl -X POST /api/predict \
  -F "excel_file=@my_bridge.xlsx" \
  -G -d "format=dxf" \
  --output bridge_drawing.dxf

❓ FAQ

What Excel format does the app accept?
Three-column format: Value (column A), Variable (column B), Description (column C). No header row required, but if present it is auto-skipped. Download any template above to see the exact format.
Which AutoCAD version should I choose?
R2010 (DXF AC1024) is recommended for AutoCAD 2010 and later, and for all modern CAD viewers. Choose R2006 (DXF AC1018) only if you are using AutoCAD 2006 or older software that cannot open R2010 files.
My drawing looks empty / wrong. What should I check?
1. Ensure NSPAN, SPAN1, LBRIDGE are consistent (LBRIDGE = NSPAN × SPAN1).
2. Check that TOPRL > DATUM (road level above river bed).
3. Verify RTL > SOFL (road top above soffit).
4. Run the Quality Check tab — it will flag the specific issue.
Can I use JSON or text files instead of Excel?
Yes. The Smart Input Processor accepts .xlsx, .xls, .csv, .json, and .txt (key=value format). Download the JSON and TXT samples above to see the expected structure.
