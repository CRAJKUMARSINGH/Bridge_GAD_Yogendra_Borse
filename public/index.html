🔌 REST API

POST an Excel file, receive a DXF drawing. Proxied via Netlify Functions → FastAPI backend.

# Generate a bridge drawing via API
curl -X POST /api/predict \
  -F "excel_file=@bridge_params.xlsx" \
  -G -d "format=dxf" \
  --output bridge_drawing.dxf

# Health check
curl /api/health
⚡

POST /api/predict

Synchronous generation. Upload Excel, download DXF/PDF/PNG immediately.

📡

POST /api/jobs

Async queue. Returns job_id. Poll status or stream via SSE.

💚

GET /api/health

Netlify function + FastAPI backend health. Returns JSON status.
